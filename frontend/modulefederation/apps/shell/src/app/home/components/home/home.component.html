Welcome to NX Module Federation Angular app

In this application we are going to work with:

We will explore the Nx Module Federation feature, which allows you to share code and components between different microfrontends. We'll start by creating a basic Nx workspace and then use the Module Federation plugin to share modules between two different applications. We'll also demonstrate how to use dynamic remote modules, which can be loaded on-demand, and show you how to configure shared libraries and runtime sharing.


{{item.title}}

We are going to use use Kafka, a distributed message broker, to build reliable and scalable distributed systems. We'll cover the basics of Kafka, including topics, partitions, and brokers, and show you how to create a simple producer and consumer using the Java API.

{{item.title}}

We will learn how to use Docker to containerize your applications. Docker is a platform that allows you to package your applications into containers, which can then be run anywhere, regardless of the underlying system. We'll cover the basics of Docker, including images, containers, Docker Compose, and show you how to create your own Docker images using Dockerfiles.
